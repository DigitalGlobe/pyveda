
pyveda: Python tools for The Veda Machine Learning API 
=======================================

pyveda is a package for working with Training Data, Models and Inference Sets in Veda.

pyveda is MIT licenced.

Installation::

    pip install pyveda

For general information on the Veda platform and API visit http://github.com/DigitalGlobe/veda.
